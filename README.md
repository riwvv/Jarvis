> Полностью локальный голосовой помощник для Windows. Работает офлайн, понимает русскую речь, управляет приложениями и системой. **Ничего не уходит в облако.**

---

## 📦 Возможности

| Область | Что умеет |
|---------|-----------|
| 🎙️ Голос | Пробуждение по слову "Джарвис", распознавание через Vosk |
| 🧠 ИИ | Локальный через Ollama (модели Qwen3.5:4b, gpt-oss:20b) |
| 💾 Память | RAG: запоминает команды и находит их по смыслу |
| 🖥️ Управление | Запуск программ, открытие сайтов, громкость, файлы |
| 🎵 Музыка | Подхват медиаконтроллера, пауза/продолжить, далее/прошлая, инфо |
| 🎨 Интерфейс | HUD-экран с анимациями, работа в трее |
| ℹ️ Контекстное меню | авто-скрытие, всегда открыт |

## 📋 Требования

- Windows 10 или 11
- .NET 8.0 или 10.0
- Ollama 0.1.17+
- 8 ГБ ОЗУ (рекомендуется 16 ГБ)
- 6 ГБ VRAM (рекомундуется 8 ГБ RTX)

## 🚀 Быстрый старт

### 1. Установите Ollama и модели

Скачайте Ollama с [официального сайта](https://ollama.com/download) и установите.

Откройте терминал и выполните:
```
ollama pull qwen3.5:4b
ollama pull qwen3-embedding:4b
```

### 2. Установите модель Vosk

Скачайте `vosk-model-ru-0.42` с [официальной страницы](https://alphacephei.com/vosk/models).

Распакуйте архив в папку `Models` рядом с `Jarvis.exe`:

```
📁 Jarvis
├── 📄 Jarvis.exe
└── 📁 Models
    └── 📁 vosk-model-ru-0.42
```

### 3. Запустите Jarvis

Скачайте последнюю версию из раздела [Releases](https://github.com/riwvv/Jarvis/releases) и запустите `Jarvis.exe`.

## 🗣️ Примеры команд

Скажите **«Джарвис»**, затем команду:

| Вы говорите | Джарвис делает |
|-------------|----------------|
| «Открой YouTube» | Открывает YouTube |
| «Открой Chrome» | Запускает браузер |
| «Сверни все окна» | Сворачивает окна |
| «Сделай громкость 50%» | Меняет громкость |
| «Как меня зовут?» | Ищет в памяти |
| «О чём я тебя сегодня просил?» | Ищет в памяти |
| «Покажи все программы» | Создаёт список приложений |

## 🧠 Как работает RAG-память

1. **Автосохранение** — после успешной команды диалог сохраняется в векторную базу
2. **Поиск** — при вопросах о прошлом LLM сама обращается к памяти
3. **Семантика** — поиск по смыслу, а не по точным словам

Хранилище векторов: `%LocalAppData%\Jarvis\vectors.json\kb.json`

Что бы очистить память и начать общение с Jarvis с нуля, удалите `vectors.json\kb.json`. Файл создаётся автоматически пустым при его отсутсвии.

## 🔧 Устранение неполадок

| Проблема | Решение |
|----------|---------|
| Ollama не отвечает | Проверьте, что Ollama запущен в трее |
| Модель не отвечает | Выполните `ollama pull qwen3.5:4b` |
| RAG не работает | Выполните `ollama pull qwen3-embedding:4b` |
| Логи | Смотрите `logs/jarvis-logs.txt` |
