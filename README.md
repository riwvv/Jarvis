> **Полностью локальный** голосовой ИИ агент для Windows. Работает офлайн, понимает русскую речь, управляет приложениями и системой.  
> 🛡️ **Ничего не уходит в облако — полная приватность.**

Jarvis превращает ваш компьютер в персонального помощника. Откройте браузер, включите музыку, создайте файл или спросите о прошлом — всё это одной фразой.

---

## ✨ Ключевые возможности

| Область | Что умеет |
|---------|-----------|
| 🎙️ **Голос** | Пробуждение по слову *"Джарвис"*, распознавание через **Vosk** (офлайн) |
| 🧠 **ИИ** | Локальный через **Ollama** (модели Qwen3.5, Eve V2, GPT-OSS). Полный контроль над данными |
| 💾 **Память** | **RAG-память**: ассистент запоминает важные факты и находит их по смыслу |
| ⚡ **Скорость** | **Мгновенные команды (<1 секунды)** для типовых действий |
| 🖥️ **Управление ПК** | Запуск программ и игр (Steam в том числе), открытие сайтов, управление громкостью и активно развивающаяся файловая система |
| 🎵 **Медиа** | Управление плеером: пауза, следующий трек, информация о треке |
| 🧠 **Умные сценарии** | Распознавание неполных команд, создание файлов из буфера обмена |
| ⛅ **Погода** | Искать погоду по названию города или по автоопределению (сегодня/завтра) |
| 🎨 **Интерфейс** | Неоновый HUD-экран с анимацией, работа в системном трее |
| 🎮 **Мини-игры** | Простые мини-игры для отвлечение от работы и развлечения |

## 📂 Файловая система

Jarvis предоставляет полноценное голосовое управление файлами в специальных папках Windows: **Downloads, Documents, Pictures, Videos, Music, Desktop**.

### Что умеет файловая система:

| Команда | Что делает |
|---------|------------|
| «Проанализируй папку загрузки» | Анализирует содержимое папки: количество файлов, папок и общий размер |
| «Открой файл отчёт.docx из документов» | Открывает файл в программе по умолчанию |
| «Создай текстовик заметка на рабочем столе» | Создаёт новый текстовый файл |
| «Перемести все фотки из загрузок на рабочий стол» | Перемещает файлы по типу в нужную папку |
| «Переименуй файл старый_отчёт в новый_отчёт в документах» | Переименовывает файл |
| «Удали файл тест.docx с рабочего стола» | Удаляет файл (с подтверждением, в корзину) |
| «Верни всё как было» | Отменяет последнее действие (контекстная память) |

### Интеллектуальный поиск

Jarvis находит файлы даже если вы ошиблись в названии. Он использует **нечёткое сравнение (расстояние Левенштейна)**, чтобы найти максимально похожий файл. Например, «открой видос попытка» найдёт файл «попытка.mp4».

## 📋 Требования

- Windows 10 или 11
- .NET 8.0 или 10.0
- Ollama 0.1.17+
- 8 ГБ ОЗУ (рекомендуется 16 ГБ)
- 6 ГБ VRAM (рекомундуется RTX 8 ГБ)

## 🚀 Быстрый старт

### 1. Установите Ollama и модели

Скачайте Ollama с [официального сайта](https://ollama.com/download) и установите.

Откройте терминал и выполните:

```bash
# Для работы "мозгов" ассистента
ollama pull jeffgreen311/Eve-V2-Unleashed-Qwen3.5-8B-Liberated-4K-4B-Merged

# Для работы RAG-памяти (поиск по смыслу)
ollama pull all-minilm:l6-v2
```

### 2. Установите модель Vosk

Скачайте `vosk-model-ru-0.42` с [официальной страницы](https://alphacephei.com/vosk/models).

Распакуйте архив в папку `Models` рядом с `Jarvis.exe`:

```
📁 Jarvis
├── 📄 Jarvis.exe
└── 📁 Models
    └── 📁 vosk-model-ru-0.42
```

### 3. Запустите Jarvis

Скачайте последнюю версию из раздела [Releases](https://github.com/riwvv/Jarvis/releases) и запустите `Jarvis.exe`.

При запуске Jarvis сообщит о начале загрузки и скажет, когда будет готов к работе. Никаких загрузочных окон.

## 🧠 Как работает RAG-память

1. **Автосохранение** — после успешной команды диалог сохраняется в векторную базу
2. **Поиск** — при вопросах о прошлом LLM сама обращается к памяти
3. **Семантика** — поиск по смыслу, а не по точным словам

Хранилище векторов: `%LocalAppData%\Jarvis\vectors.json\kb.json`

Что бы очистить память и начать общение с Jarvis с нуля, удалите `vectors.json\kb.json`. Файл создаётся автоматически пустым при его отсутсвии.
