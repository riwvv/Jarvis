# 🎙️ Jarvis — Голосовой ассистент на C# с локальным ИИ

[![GitHub stars](https://img.shields.io/github/stars/riwvv/Jarvis)](https://github.com/riwvv/Jarvis/stargazers)

[![WPF](https://img.shields.io/badge/WPF-.NET%2010.0-green)](https://learn.microsoft.com/ru-ru/dotnet/desktop/wpf/)

> **Интеллектуальный голосовой помощник для Windows с полной приватностью.**  
> Работает офлайн, понимает русскую речь, управляет приложениями и системой.  
> **Всё локально — ничего не уходит в облако.**

Jarvis — это полноценный голосовой ассистент, который не требует интернета. Он пробуждается по слову **«Джарвис»**, распознаёт голосовые команды (Vosk), понимает их с помощью локального ИИ (Ollama + Qwen/GPT-OSS) и выполняет действия: открывает программы, сайты, управляет громкостью и отвечает на вопросы.

---

## ✨ Ключевые возможности

### 🎯 Голосовое управление
-   **Пробуждение по ключевому слову** — просто скажите «Джарвис».
-   **Распознавание речи** — офлайн через Vosk (русский язык).
-   **Голосовая активность** — ассистент сам понимает, когда вы закончили команду.

### 🧠 ИИ и автоматизация
-   **Локальный LLM** — поддержка Ollama. Используйте модели Qwen, GPT-OSS, Llama.
-   **Semantic Kernel** — умные цепочки действий и контекстные диалоги.
-   **Долговременная память** — ассистент помнит вас и прошлые разговоры (LiteDB).

### 🖥️ Управление системой
-   **Запуск приложений** — «Открой Visual Studio», «Запусти Chrome».
-   **Управление громкостью** — «Сделай громче 50%», «Выключи звук».
-   **Открытие сайтов** — «Открой YouTube», «Перейди на github.com».
-   **Работа с файлами** — «Покажи все программы» (создаёт список на рабочем столе).

### 🎨 Интерфейс
-   **Стильный HUD-экран** с киберпанк-дизайном.
-   **Анимированные переходы** состояния (SLEEP → LISTENING → PROCESSING → EXECUTE).
-   **Прозрачное окно** с эффектами свечения и перемещением мышью.
-   **Работа в трее** — ассистент не мешает и легко вызывается.

## 📋 Требования к системе

| Компонент | Минимальная версия |
| :--- | :--- |
| **Windows** | 10 / 11 |
| **.NET Runtime** | 8.0 или 10.0 |
| **Ollama** | 0.1.17+ (для ИИ) |
| **ОЗУ** | 8 ГБ (рекомендуется 16 ГБ для моделей 20B) |
| **CPU** | x64, 2+ ядра |
| **SSD** | Рекомендуется для быстрой загрузки модели |

**Важно:** Для работы Jarvis не нужен интернет. Всё обрабатывается локально.

## 🚀 Быстрый старт (пошаговая инструкция)

### 1. Установите и запустите Ollama
Ollama нужен для работы языковой модели (LLM).

1.  Скачайте Ollama с [официального сайта](https://ollama.com/download).
2.  Установите его, оставив настройки по умолчанию.
3.  После установки Ollama автоматически запустится в фоне (убедитесь, что он работает — иконка в трее).
4.  Откройте терминал (PowerShell / CMD) и скачайте одну из моделей (на выбор):
    ```bash
    # Лёгкая и быстрая модель
    ollama pull llama3.2:3b

    # Более мощная модель (требует 16 ГБ ОЗУ)
    ollama pull SimonPu/gpt-oss:20b_Q4_K_M
    ```
    Дождитесь окончания загрузки (это займёт время).

### 2. Установите модель для распознавания речи (Vosk)
1.  Скачайте русскую модель с [официальной страницы Vosk](https://alphacephei.com/vosk/models):
    *   `vosk-model-ru-0.42` (рекомендуемая, около 1.2 ГБ).
2.  Распакуйте архив.
3.  Переименуйте полученную папку в `vosk-model-ru-0.42`.
4.  Поместите её в папку `Models` рядом с `Jarvis.exe`.
    ```
    📁 Jarvis
       📄 Jarvis.exe
       📁 Models
          📁 vosk-model-ru-0.42
    ```

### 3. Запустите Jarvis
1.  Скачайте последнюю версию `Jarvis.exe` из раздела [Releases](https://github.com/riwvv/Jarvis/releases) (его пока нет, ссылка будет после публикации).
2.  Запустите `Jarvis.exe`.
3.  При первом запуске может потребоваться разрешение брандмауэра — разрешите. Это нужно для связи с Ollama.
4.  Если появится окно с ошибкой **«Ollama не запущена»**, вернитесь к шагу 1 и убедитесь, что Ollama работает.

## 🗣️ Примеры команд

После запуска скажите **«Джарвис»** и дождитесь сигнала (или просто отдайте команду).

| Вы говорите | Джарвис делает |
| :--- | :--- |
| «Открой Chrome» | Запускает браузер. |
| «Как меня зовут?» | Вспоминает, если вы называли имя ранее. |
| «Сверни все окна» | Сворачивает все окна на рабочем столе. |
| «Сделай громче 50 процентов» | Изменяет системную громкость. |
| «Открой YouTube» | Открывает сайт в браузере по умолчанию. |
| «Покажи все программы» | Создаёт на рабочем столе файл со списком установленных приложений. |
