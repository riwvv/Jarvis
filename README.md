# 🎙️ Jarvis — Голосовой ассистент на C# с локальным ИИ

> Интеллектуальный голосовой помощник для Windows с полной приватностью.  
> Работает офлайн, понимает русскую речь, управляет приложениями и системой.

---

## ✨ Возможности

### 🎯 Голосовое управление
- **Пробуждение по ключевому слову** "Джарвис"
- **Распознавание речи** через Vosk (офлайн, русский язык)
- **Голосовая активность** для определения окончания команды

### 🧠 ИИ и автоматизация
- **Локальный LLM** (Ollama / Qwen2.5) — полная приватность данных
- **Semantic Kernel** — умные цепочки действий
- **Контекстные диалоги** — ассистент помнит предыдущие команды

### 🖥️ Управление системой
- **Запуск приложений** по голосу ("Открой Visual Studio", "Запусти Chrome")
- **Управление громкостью** ("Сделай громче 50%", "Выключи звук")
- **Открытие сайтов** ("Открой YouTube", "Перейди на github.com")
- **Список установленных программ** ("Покажи все программы")

### 🎨 Интерфейс
- **Стильный HUD-экран** с киберпанк-дизайном
- **Анимированные переходы** состояния (SLEEP → LISTENING → PROCESSING → EXECUTE)
- **Прозрачное окно** с эффектами свечения
- **Перемещение окна** мышью

## 📋 Требования

| Компонент | Минимальная версия |
|-----------|-------------------|
| Windows | 10 / 11 |
| .NET Runtime | 8.0 или 10.0 |
| Ollama | 0.1.17+ (для LLM) |
| ОЗУ | 4 ГБ (8 ГБ рекомендуется) |
| CPU | x64, 2+ ядра |

## 🚀 Быстрый старт

### 1. Установка Ollama и модели

```bash
# Скачайте Ollama с официального сайта
# https://ollama.com/download

# Запустите Ollama и скачайте русскоязычную модель

# Убедитесь, что Ollama запущен (порт 11434)
ollama serve
